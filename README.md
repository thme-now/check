# QR-Code Validierungs-App

Zentrale Validierung von Produktnummern über QR-Code mit GitHub-gehosteter CSV-Datenbank.

## Funktionen
- Automatischer CSV-Import von GitHub
- Globale Verfügbarkeit 24/7
- QR-Code Integration
- Echtzeit-Validierung
- Mobile-optimiert

## CSV-Datei aktualisieren
Bearbeiten Sie `nummern.csv` direkt auf GitHub - Änderungen sind sofort verfügbar!

## Verwendung für Kunden
QR-Code scannen → Automatische Validierung → Ergebnis
